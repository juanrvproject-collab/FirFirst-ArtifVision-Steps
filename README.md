# First-ArtifVision-Steps
My first project of Artificial Vision in python usion OpenCV


In this code I used an OpenCV library to create a face detection system in video by applying a pre-trained Haar Cascade classifier. It iterates through the video frames, converting them to grayscale for work with them.
